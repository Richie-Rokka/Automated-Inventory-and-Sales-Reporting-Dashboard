## 📌 Project Overview

This project showcases the development of an **automated data analytics solution** using Excel to streamline reporting, improve data accuracy, and deliver actionable business insights.

By leveraging **Power Query, structured data modeling, and interactive dashboards**, the solution transforms raw operational data into a scalable reporting system that supports decision-making across sales and inventory functions.

## ✨ Features

- Automated data transformation using Power Query  
- Multi-source data integration (Sales, Inventory, Reference)  
- Interactive dashboard with slicers  
- KPI tracking and trend analysis  
- Inventory risk detection (Low Stock Alerts)  
- One-click refresh for real-time updates

---

## 🎯 Problem Statement

Organizations often rely on manual reporting processes that:

- Require repetitive data consolidation from multiple sources  
- Are prone to inconsistencies and errors  
- Delay insight generation for business stakeholders  
- Lack scalability for growing datasets  

In this case, sales and inventory data were siloed, requiring manual merging and analysis, limiting visibility into performance and operational risks.

---

## ⚙️ Approach & Methodology

### 🔹 Data Integration & Preparation
- Consolidated data from multiple sources (sales, inventory, product reference)
- Standardized formats and ensured data consistency
- Structured datasets for scalable analysis

---

### 🔹 Data Transformation (Power Query)
- Built automated ETL workflows:
  - Data cleaning (removing duplicates, handling nulls)
  - Type standardization  
  - Multi-table joins using SKU as a primary key  
- Resolved one-to-many join issues using **aggregation (Group By)**
- Created derived fields:
  - Stock Status  
  - Profit & Revenue Metrics  
  - Product categorization  

---

### 🔹 Data Modeling & Analysis
- Designed a layered data model:
  - Raw → Clean → Analytical  
- Developed pivot tables to analyze:
  - Revenue trends  
  - Regional performance  
  - Product/category performance  
  - Inventory risk  

---

### 🔹 Dashboard & Visualization
- Built an interactive dashboard with:
  - KPI tracking (Revenue, Orders, Profit, Low Stock Items)  
  - Time-series analysis  
  - Regional and category breakdowns  
  - Inventory alerts  

- Implemented **slicers** for dynamic filtering:
  - Region  
  - Category  
  - Time
